# 1. 介绍

该项目为 Micronaut 实现了一个项目生成器，它既是一个命令行（CLI）应用程序，也是一个 API，可以作为 HTTP 服务器运行、部署到 Servlet 容器或在无服务器平台（Google Cloud Run、Google Cloud Function、AWS Lambda 或 Azure Function）上执行。
